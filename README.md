# Manual de Uso - Ejercicios Prolog

Este proyecto contiene los ejercicios del documento **Prolog - Ejercicios.pdf**, pensados para practicar lógica computacional con el lenguaje **Prolog**.

## Requisitos

- Tener instalado **SWI-Prolog** en tu computadora.
- Contar con un editor de texto (VSCode, Notepad++, etc.).
- Conocimientos básicos de hechos, reglas y consultas en Prolog.
